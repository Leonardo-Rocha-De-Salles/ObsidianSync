La funzione parte intera indicata con $[x]$ è definita da:
$$[x] = quell'intero \ n \ tale \ che \ n \leq x <n+1$$
$$[2.37] = 2$$
$$[3] = 3$$
$$[-1.8] = -2$$
![[Pasted image 20240724180812.png]]
La mantissa invece è una funzione indicata con (x), definita come :
$$(x) = x - [x]$$
$$(2,38) = 0,38$$
$$(3) = 0$$
$$(-1,8) = 0,2$$
